Voltaje

El voltaje, también conocido como tensión eléctrica o diferencia de potencial, es una magnitud física que cuantifica la diferencia de potencial eléctrico entre dos puntos. Es la fuerza que impulsa a los electrones a través de un conductor, similar a la presión en un sistema hidráulico.

Tipos
Medición
Aplicaciones

¿Qué es el Voltaje?

El voltaje es la diferencia de potencial eléctrico entre dos puntos de un circuito. Se mide en voltios (V) y representa la energía por unidad de carga que se necesita para mover una carga eléctrica entre esos dos puntos.

Definición formal: El voltaje entre dos puntos es el trabajo por unidad de carga necesario para mover una carga eléctrica desde un punto a otro, contra el campo eléctrico.

Analogía Hidráulica

Una forma sencilla de entender el voltaje es compararlo con la presión del agua en una tubería:

  • El voltaje es como la presión del agua
  • La corriente es como el flujo de agua
  • Una batería es como una bomba de agua
  • Mayor voltaje = mayor presión = mayor capacidad para realizar trabajo

Fórmula Básica

V = W / Q

Donde:

  • V: Voltaje (voltios, V)
  • W: Trabajo o energía (julios, J)
  • Q: Carga eléctrica (culombios, C)

Tipos de Voltaje

Voltaje de Corriente Continua (DC)

El voltaje de corriente continua (DC) mantiene una polaridad constante a lo largo del tiempo. La dirección del flujo de corriente no cambia.

Características:
  • Valor constante en el tiempo
  • Polaridad fija (positivo y negativo)
  • Representado por una línea recta en un osciloscopio
Fuentes comunes:
  • Baterías y pilas
  • Paneles solares
  • Fuentes de alimentación DC
Voltaje de Corriente Alterna (AC)

El voltaje de corriente alterna (AC) cambia su polaridad periódicamente con el tiempo, siguiendo generalmente una forma de onda senoidal.

Características:
  • Valor y polaridad variables en el tiempo
  • Frecuencia medida en hercios (Hz)
  • Forma de onda típicamente senoidal
Parámetros importantes:
  • Valor pico (Vp): Amplitud máxima
  • Valor pico a pico (Vpp): Diferencia entre el valor máximo positivo y negativo
  • Valor eficaz o RMS (Vrms): Equivalente DC que produciría la misma potencia (Vrms = Vp/√2 para onda senoidal)
Voltaje Pulsante

El voltaje pulsante alterna entre un valor fijo y cero (o entre dos valores fijos) a intervalos regulares o irregulares.

Tipos comunes:
  • Onda cuadrada: Alterna entre dos niveles con transiciones rápidas
  • PWM (Modulación por Ancho de Pulso): Onda cuadrada con ciclo de trabajo variable
  • Pulsos únicos: Señales transitorias de corta duración
Aplicaciones:
  • Control de motores
  • Comunicaciones digitales
  • Sistemas de control
  • Convertidores DC-DC

Medición del Voltaje

La medición del voltaje es una de las operaciones más comunes en electricidad y electrónica. Se realiza con un instrumento llamado voltímetro, que actualmente suele formar parte de un multímetro.

Instrumentos de Medición

  • Multímetro Digital: El instrumento más común para medir voltaje, corriente y resistencia.
  • Osciloscopio: Muestra gráficamente la forma de onda del voltaje en función del tiempo.
  • Probador de Voltaje: Dispositivo simple que indica la presencia de voltaje.
  • Analizador de Potencia: Mide voltaje, corriente y otros parámetros en sistemas eléctricos.

Procedimiento para Medir Voltaje

  1. Seleccionar la función: Gire el selector del multímetro a la posición de voltaje (V).
  2. Seleccionar AC o DC: Elija el tipo de voltaje que va a medir (⎓ para DC, ∼ para AC).
  3. Seleccionar el rango: Si el multímetro no es autorangos, seleccione un rango superior al voltaje esperado.
  4. Conectar las puntas: Conecte la punta negra al terminal COM y la roja al terminal V.
  5. Medir en paralelo: Conecte las puntas en paralelo con el componente o circuito a medir.
  6. Leer el valor: Observe la lectura en la pantalla del multímetro.

Importante: Al medir voltaje, el multímetro debe conectarse siempre en paralelo con el componente o circuito. Nunca conecte un voltímetro en serie, ya que puede dañar el instrumento o el circuito.

Consideraciones de Seguridad

Precauciones
  • Verifique que el multímetro esté configurado para el tipo correcto de voltaje (AC o DC)
  • Utilice el rango adecuado para evitar daños al instrumento
  • Mantenga las manos detrás de las protecciones de las puntas de prueba
  • No mida voltajes superiores a los especificados para el multímetro
  • Tenga especial cuidado al medir voltajes superiores a 30V AC o 60V DC
Buenas Prácticas
  • Utilice una sola mano cuando sea posible para evitar que la corriente atraviese el corazón
  • Trabaje sobre superficies secas y aisladas
  • Verifique que las puntas de prueba estén en buen estado
  • Desconecte la alimentación antes de realizar conexiones en circuitos de alta tensión
  • Utilice equipo de protección personal cuando sea necesario

Aplicaciones del Voltaje

El voltaje es un concepto fundamental en electricidad y electrónica, con innumerables aplicaciones en la vida cotidiana y en la industria.

Suministro Eléctrico

El voltaje es fundamental en la generación, transmisión y distribución de energía eléctrica. Los diferentes niveles de voltaje permiten transportar la electricidad de manera eficiente.

  • Alta tensión: 110kV-800kV (transmisión)
  • Media tensión: 1kV-36kV (distribución)
  • Baja tensión: 120V-480V (consumo)

Electrónica

Los dispositivos electrónicos funcionan con diferentes niveles de voltaje, desde microvoltios hasta cientos de voltios.

  • Microprocesadores: 1.2V-3.3V
  • Lógica digital: 3.3V, 5V
  • Circuitos analógicos: ±12V, ±15V
  • Pantallas: 5V-30V

Aplicaciones Específicas

Control y Automatización

El voltaje se utiliza como señal de control en sistemas automáticos:

  • Señales de control: 0-10V, 4-20mA (convertida a voltaje)
  • Control de velocidad: Variadores de frecuencia con señales de 0-10V
  • Posicionamiento: Servomotores controlados por voltaje
  • PLC: Entradas y salidas analógicas de voltaje
Comunicaciones

Las señales de comunicación utilizan diferentes niveles de voltaje:

  • RS-232: ±3V a ±15V
  • RS-485: ±1.5V a ±6V
  • USB: 5V, con señales diferenciales de ±0.2V
  • Ethernet: Señales diferenciales de bajo voltaje
  • Fibra óptica: Señales ópticas convertidas a/desde voltaje

Corriente

La corriente eléctrica es el flujo ordenado de cargas eléctricas (generalmente electrones) a través de un conductor. Es una magnitud física fundamental en electricidad y electrónica, y se mide en amperios (A).

¿Qué es la Corriente Eléctrica?

La corriente eléctrica es el movimiento ordenado de cargas eléctricas a través de un material conductor. En los metales, estas cargas son electrones libres que se desplazan debido a la acción de un campo eléctrico.

Definición formal: La corriente eléctrica es la tasa de flujo de carga eléctrica que pasa por un punto determinado de un circuito eléctrico por unidad de tiempo.

Analogía Hidráulica

Al igual que con el voltaje, podemos entender la corriente eléctrica mediante una analogía con el flujo de agua:

  • La corriente eléctrica es como el caudal de agua en una tubería
  • El voltaje es como la presión que impulsa el agua
  • La resistencia es como el diámetro de la tubería (restricción al flujo)
  • Mayor corriente = mayor flujo de electrones = mayor capacidad para realizar trabajo

Fórmula Básica

I = Q / t

Donde:

  • I: Corriente (amperios, A)
  • Q: Carga eléctrica (culombios, C)
  • t: Tiempo (segundos, s)

Un amperio (A) equivale al paso de un culombio de carga por segundo.

Ley de Ohm

La relación entre corriente, voltaje y resistencia viene dada por la Ley de Ohm:

I = V / R

Donde:

  • I: Corriente (amperios, A)
  • V: Voltaje (voltios, V)
  • R: Resistencia (ohmios, Ω)

Tipos de Corriente Eléctrica

Corriente Continua (DC)

La corriente continua (DC) fluye en una sola dirección. Los electrones se mueven constantemente del polo negativo al positivo de la fuente de alimentación.

Características:
  • Dirección constante del flujo de electrones
  • Valor constante o con pequeñas variaciones
  • No cambia de polaridad con el tiempo
Fuentes comunes:
  • Baterías y pilas
  • Células solares
  • Fuentes de alimentación DC
  • Generadores DC
Corriente Alterna (AC)

La corriente alterna (AC) cambia periódicamente su dirección y magnitud con el tiempo, generalmente siguiendo una forma de onda senoidal.

Características:
  • Dirección y magnitud variables en el tiempo
  • Frecuencia medida en hercios (Hz)
  • Forma de onda típicamente senoidal
Parámetros importantes:
  • Valor pico (Ip): Amplitud máxima
  • Valor pico a pico (Ipp): Diferencia entre el valor máximo positivo y negativo
  • Valor eficaz o RMS (Irms): Equivalente DC que produciría el mismo efecto térmico (Irms = Ip/√2 para onda senoidal)
Característica Corriente Continua (DC) Corriente Alterna (AC)
Dirección del flujo Unidireccional Bidireccional (cambia periódicamente)
Forma de onda Constante o con pequeñas variaciones Generalmente senoidal
Frecuencia 0 Hz (no cambia) 50/60 Hz (doméstica), variable en aplicaciones específicas
Aplicaciones típicas Electrónica, dispositivos portátiles, vehículos Distribución eléctrica, electrodomésticos, industria

Medición de la Corriente

La medición de corriente eléctrica es fundamental en electricidad y electrónica para verificar el funcionamiento de circuitos, diagnosticar problemas y garantizar la seguridad.

Instrumentos de Medición

  • Amperímetro: Instrumento específico para medir corriente eléctrica.
  • Multímetro: Dispositivo versátil que puede medir corriente, voltaje y resistencia.
  • Pinza Amperimétrica: Permite medir corriente sin interrumpir el circuito.
  • Osciloscopio: Muestra gráficamente la forma de onda de la corriente usando una resistencia shunt.

Procedimiento para Medir Corriente

  1. Seleccionar la función: Gire el selector del multímetro a la posición de corriente (A).
  2. Seleccionar AC o DC: Elija el tipo de corriente que va a medir (⎓ para DC, ∼ para AC).
  3. Seleccionar el rango: Si el multímetro no es autorangos, seleccione un rango superior a la corriente esperada.
  4. Interrumpir el circuito: Apague la alimentación y abra el circuito en el punto donde desea medir.
  5. Conectar en serie: Conecte el multímetro en serie con el circuito (la corriente debe pasar a través del multímetro).
  6. Alimentar el circuito: Encienda la alimentación y lea el valor en la pantalla.

Importante: A diferencia del voltaje, la corriente se mide siempre en serie con el circuito. Nunca conecte un amperímetro en paralelo, ya que su baja resistencia interna provocaría un cortocircuito que podría dañar el instrumento y el circuito.

Efectos de la Corriente Eléctrica

La corriente eléctrica produce diversos efectos físicos que son la base de numerosas aplicaciones tecnológicas. Estos efectos dependen principalmente de la intensidad de la corriente, su tipo (AC/DC) y el medio por el que circula.

Efecto Térmico (Joule)

Cuando una corriente eléctrica circula por un conductor, parte de la energía cinética de los electrones se transforma en calor debido a los choques con los átomos del material.

P = I² × R
Aplicaciones:
  • Calefactores eléctricos
  • Hornos y cocinas eléctricas
  • Soldadores eléctricos
  • Fusibles

Efecto Magnético

Toda corriente eléctrica genera un campo magnético a su alrededor. La intensidad del campo es proporcional a la corriente.

B ∝ I
Aplicaciones:
  • Electroimanes
  • Motores eléctricos
  • Relés y contactores
  • Transformadores
Efecto Fisiológico

La corriente eléctrica puede afectar al cuerpo humano de diferentes maneras según su intensidad:

Intensidad (mA) Efecto en el cuerpo humano
0-1 Imperceptible
1-5 Sensación leve, cosquilleo
5-10 Choque doloroso, contracciones musculares
10-30 Contracciones musculares fuertes, dificultad para soltar
>50 Fibrilación ventricular, paro cardíaco, quemaduras

Seguridad: La corriente eléctrica puede ser peligrosa. Incluso corrientes relativamente pequeñas pueden ser letales si atraviesan órganos vitales como el corazón. Siempre tome precauciones al trabajar con electricidad y respete las normas de seguridad.

Potencia

La potencia eléctrica es la cantidad de energía que se transfiere, consume o disipa por unidad de tiempo en un circuito eléctrico. Es una magnitud fundamental que determina la capacidad de realizar trabajo eléctrico y se mide en vatios (W).

¿Qué es la Potencia Eléctrica?

La potencia eléctrica es la tasa a la que se consume o transfiere energía en un circuito eléctrico. Representa la rapidez con la que se realiza un trabajo eléctrico.

Definición formal: La potencia eléctrica es la cantidad de energía eléctrica transferida o transformada por unidad de tiempo. Se mide en vatios (W), donde 1 vatio equivale a 1 julio por segundo (1 W = 1 J/s).

Fórmula Básica

P = E / t

Donde:

  • P: Potencia (vatios, W)
  • E: Energía (julios, J)
  • t: Tiempo (segundos, s)

Fórmulas en Circuitos Eléctricos

P = V × I (Circuitos DC)
P = V × I × cos φ (Circuitos AC)

Donde:

  • P: Potencia (vatios, W)
  • V: Voltaje (voltios, V)
  • I: Corriente (amperios, A)
  • cos φ: Factor de potencia (en circuitos AC)

Tipos de Potencia Eléctrica

En circuitos de corriente continua (DC), solo existe un tipo de potencia. Sin embargo, en circuitos de corriente alterna (AC), existen diferentes tipos de potencia que es importante distinguir.

Potencia Activa (P)

La potencia activa, también llamada potencia real o potencia útil, es la que realmente se transforma en trabajo o calor. Es la potencia que realmente "hace el trabajo" en el circuito.

Características:
  • Se mide en vatios (W), kilovatios (kW), etc.
  • Es la única potencia que realiza trabajo útil
  • Es la potencia que se factura principalmente en el consumo eléctrico
Potencia Reactiva (Q)

La potencia reactiva es la potencia que oscila entre la fuente y la carga debido a elementos reactivos (inductivos o capacitivos). No realiza trabajo útil, pero es necesaria para crear campos magnéticos y eléctricos.

Características:
  • Se mide en voltamperios reactivos (VAr)
  • No produce trabajo útil
  • Oscila entre la fuente y la carga
  • Necesaria para el funcionamiento de motores, transformadores, etc.
Potencia Aparente (S)

La potencia aparente es la suma vectorial de la potencia activa y la potencia reactiva. Representa la potencia total que suministra la fuente de alimentación.

Características:
  • Se mide en voltamperios (VA)
  • Es la potencia total suministrada
  • Determina el dimensionamiento de equipos y conductores

Triángulo de Potencias

El triángulo de potencias es una representación gráfica de la relación entre los tres tipos de potencia en sistemas de corriente alterna:

  • La potencia activa (P) se representa en el eje horizontal
  • La potencia reactiva (Q) se representa en el eje vertical
  • La potencia aparente (S) es la hipotenusa del triángulo
  • El ángulo φ es el ángulo de desfase entre tensión y corriente

A partir del triángulo de potencias se pueden establecer las siguientes relaciones:

  • S = √(P² + Q²)
  • P = S × cos φ
  • Q = S × sin φ
  • tan φ = Q / P

Factor de Potencia

Factor de Potencia (FP): Es la relación entre la potencia activa (P) y la potencia aparente (S). Indica qué fracción de la potencia total suministrada se convierte en trabajo útil.

FP = P / S = cos φ

Cálculo de la Potencia Eléctrica

El cálculo de la potencia eléctrica varía según el tipo de circuito (DC o AC) y los datos disponibles.

Cálculo en Circuitos DC

En circuitos de corriente continua, la potencia se puede calcular de varias formas según los datos disponibles:

Fórmula Variables Cuando usarla
P = V × I V: Voltaje (V)
I: Corriente (A)		 Cuando se conocen el voltaje y la corriente
P = I² × R I: Corriente (A)
R: Resistencia (Ω)		 Cuando se conocen la corriente y la resistencia
P = V² / R V: Voltaje (V)
R: Resistencia (Ω)		 Cuando se conocen el voltaje y la resistencia

Ejemplo: Un circuito tiene una resistencia de 100 Ω y está conectado a una fuente de 12 V. La potencia será:

P = V² / R = 12² / 100 = 144 / 100 = 1.44 W

I: Corriente RMS (A)

Aplicaciones de la Potencia Eléctrica

La potencia eléctrica es un concepto fundamental con numerosas aplicaciones prácticas en diversos campos.

Diseño de Sistemas Eléctricos

El cálculo de potencia es fundamental para dimensionar correctamente:

  • Conductores y cables
  • Protecciones (fusibles, interruptores)
  • Transformadores
  • Generadores y fuentes de alimentación

Eficiencia Energética

La potencia es clave para evaluar y mejorar la eficiencia:

  • Auditorías energéticas
  • Corrección del factor de potencia
  • Selección de equipos eficientes
  • Reducción de pérdidas

Dato importante: En la industria, mejorar el factor de potencia de 0.7 a 0.95 puede reducir las pérdidas en conductores hasta en un 45% y liberar capacidad en transformadores y generadores.

Resistencias

Las resistencias son componentes electrónicos pasivos que limitan o regulan el flujo de corriente eléctrica en un circuito. Son uno de los componentes más fundamentales y ampliamente utilizados en electrónica.

Fundamentos de las Resistencias

La resistencia eléctrica es la oposición que presenta un material al paso de la corriente eléctrica. Se mide en ohmios (Ω) y está definida por la Ley de Ohm.

Ley de Ohm: La corriente que circula por un conductor es directamente proporcional a la tensión aplicada e inversamente proporcional a la resistencia.

R = V / I

Donde R es la resistencia (Ω), V es la tensión (V) e I es la corriente (A).

Factores que Afectan la Resistencia

Propiedades del Material
  • Resistividad (ρ): Propiedad intrínseca del material que indica su oposición al flujo de corriente
  • Temperatura: La mayoría de los materiales aumentan su resistencia con la temperatura (coeficiente térmico positivo)
  • Composición: Aleaciones y dopajes modifican la resistividad
Geometría del Conductor
  • Longitud (L): La resistencia es directamente proporcional a la longitud
  • Sección transversal (A): La resistencia es inversamente proporcional al área de la sección
R = ρ × L / A

Parámetros Importantes

Tolerancia

La tolerancia indica la desviación máxima permitida del valor nominal de la resistencia:

  • ±1%: Resistencias de precisión (banda marrón)
  • ±2%: Resistencias de alta precisión (banda roja)
  • ±5%: Resistencias estándar (banda dorada)
  • ±10%: Resistencias de baja precisión (banda plateada)
  • ±20%: Resistencias de muy baja precisión (sin banda)

Ejemplo: Una resistencia de 1000 Ω con tolerancia del 5% puede tener un valor real entre 950 Ω y 1050 Ω.

Potencia Nominal

La potencia nominal indica la máxima potencia que puede disipar la resistencia sin dañarse:

  • 1/8 W: Resistencias miniatura para circuitos de baja potencia
  • 1/4 W: Tamaño estándar para electrónica general
  • 1/2 W: Aplicaciones de potencia media
  • 1 W, 2 W, 5 W: Aplicaciones de potencia
  • 10 W y superiores: Resistencias de potencia con disipador

La potencia disipada se calcula como:

P = I² × R = V² / R

Tipos de Resistencias

Existen diversos tipos de resistencias que se clasifican según su construcción, materiales y aplicaciones específicas.

Resistencias de Composición de Carbón

Características:
  • Fabricadas con polvo de carbón y aglutinante
  • Económicas y versátiles
  • Tolerancia típica: ±5% a ±20%
  • Rango: 1 Ω a 22 MΩ

Resistencias de Película Metálica

Características:
  • Película delgada de metal sobre cerámica
  • Alta precisión y estabilidad
  • Tolerancia típica: ±0.1% a ±2%
  • Rango: 0.1 Ω a 10 MΩ

Resistencias Variables y Especiales

Potenciómetros y Reóstatos

Los potenciómetros son resistencias variables con tres terminales que funcionan como divisores de tensión ajustables.

Tipos principales:
  • Rotatorios: Control mediante giro (volumen, tono)
  • Deslizantes: Control lineal (ecualizadores, mezcladores)
  • Presets/Trimmers: Ajuste ocasional con destornillador
Termistores

Los termistores son resistencias cuyo valor cambia significativamente con la temperatura, utilizados para medición y control térmico.

Tipos principales:
  • NTC (Negative Temperature Coefficient): La resistencia disminuye al aumentar la temperatura
  • PTC (Positive Temperature Coefficient): La resistencia aumenta al aumentar la temperatura

Código de Colores de Resistencias

El código de colores es un sistema estandarizado para identificar el valor de las resistencias mediante bandas de colores impresas en su cuerpo. Este sistema permite una rápida identificación visual sin necesidad de instrumentos de medición.

Interpretación de las Bandas

  • Resistencias de 4 bandas: 1ª y 2ª bandas (valor), 3ª banda (multiplicador), 4ª banda (tolerancia)
  • Resistencias de 5 bandas: 1ª, 2ª y 3ª bandas (valor), 4ª banda (multiplicador), 5ª banda (tolerancia)
  • Resistencias de 6 bandas: Como las de 5 bandas, más 6ª banda (coeficiente de temperatura)

Aplicaciones de las Resistencias

Las resistencias son componentes fundamentales en electrónica y se utilizan en una amplia variedad de aplicaciones.

Limitación de Corriente

Una de las aplicaciones más comunes de las resistencias es limitar la corriente en un circuito:

  • Protección de LEDs y otros componentes sensibles
  • Establecimiento de corrientes de polarización en transistores
  • Limitación de corriente de carga en condensadores
  • Protección contra cortocircuitos

Ejemplo: Para un LED que requiere 20mA y tiene una caída de tensión de 2V, con una fuente de 5V, la resistencia necesaria sería:

R = (5V - 2V) / 0.02A = 150Ω

Divisores de Tensión

Los divisores de tensión permiten obtener una tensión de salida proporcional a la tensión de entrada:

Vout = Vin × (R2 / (R1 + R2))

Aplicaciones:

  • Acondicionamiento de señales
  • Polarización de transistores
  • Referencia de tensión
  • Medición de tensiones elevadas
Redes de Resistencias

Las redes de resistencias se utilizan para crear circuitos más complejos:

  • Redes en serie: Rtotal = R1 + R2 + R3 + ...
  • Redes en paralelo: 1/Rtotal = 1/R1 + 1/R2 + 1/R3 + ...
  • Redes en escalera: Utilizadas en conversores D/A
  • Puentes de Wheatstone: Para mediciones precisas

Consejo práctico: Al diseñar circuitos, es recomendable utilizar valores de resistencias de la serie E12 (10, 12, 15, 18, 22, 27, 33, 39, 47, 56, 68, 82) o E24, ya que son los más comunes y fáciles de conseguir.

Términos de Electricidad

Conceptos Fundamentales
Carga Eléctrica
Propiedad física de la materia que causa que ejerza y experimente fuerzas electromagnéticas. Se mide en culombios (C).

Flujo de carga eléctrica a través de un conductor. Se mide en amperios (A).

Resistencia
Oposición que presenta un material al paso de la corriente eléctrica. Se mide en ohmios (Ω).
Potencia
Cantidad de energía transferida o transformada por unidad de tiempo. Se mide en vatios (W).
Energía Eléctrica
Capacidad de realizar un trabajo mediante electricidad. Se mide en julios (J) o kilovatios-hora (kWh).
Leyes Fundamentales
Ley de Ohm
La corriente que circula por un conductor es directamente proporcional a la tensión e inversamente proporcional a la resistencia. I = V / R
Leyes de Kirchhoff
  • Ley de Corrientes: La suma de las corrientes que entran a un nodo es igual a la suma de las corrientes que salen.
  • Ley de Tensiones: La suma algebraica de las tensiones en un lazo cerrado es igual a cero.
Ley de Joule
La energía disipada en forma de calor por un conductor es proporcional al cuadrado de la corriente, a la resistencia y al tiempo. E = I² × R × t
Ley de Coulomb
La fuerza entre dos cargas eléctricas es directamente proporcional al producto de las cargas e inversamente proporcional al cuadrado de la distancia entre ellas.
Circuitos Eléctricos
Circuito Eléctrico
Conjunto de elementos conectados entre sí por los que puede circular una corriente eléctrica.
Circuito Serie
Circuito en el que los componentes están conectados uno a continuación del otro, formando un único camino para la corriente.
Circuito Paralelo
Circuito en el que los componentes están conectados entre los mismos dos puntos, formando múltiples caminos para la corriente.
Cortocircuito
Conexión de baja impedancia entre dos puntos de un circuito que normalmente están a diferentes potenciales.
Circuito Abierto
Circuito en el que se ha interrumpido el camino para la corriente, impidiendo su circulación.

Introducción a Arduino

Arduino es una plataforma de creación de electrónica de código abierto, la cual está basada en hardware y software libre, flexible y fácil de utilizar para los creadores y desarrolladores.

¿Qué es Arduino? Arduino es una plataforma de desarrollo basada en una placa electrónica con un microcontrolador y un entorno de desarrollo (IDE), diseñada para facilitar el uso de la electrónica en proyectos multidisciplinares.

Características Principales

  • Hardware libre: Los diseños de las placas están disponibles bajo licencia de código abierto
  • Software libre: El IDE y las bibliotecas son de código abierto
  • Multiplataforma: El software funciona en Windows, macOS y Linux
  • Entorno sencillo: Fácil de usar para principiantes, pero flexible para usuarios avanzados
  • Bajo costo: Las placas Arduino son relativamente económicas
  • Comunidad activa: Gran comunidad de usuarios que comparten conocimientos y proyectos
  • Extensible: Posibilidad de añadir shields (placas de expansión)
  • Variedad de placas: Diferentes modelos para distintas necesidades
  • Programación sencilla: Basada en C/C++, pero simplificada
  • Documentación amplia: Tutoriales, ejemplos y referencias disponibles

Historia y Evolución

Arduino nació en 2005 en el Instituto de Diseño Interactivo de Ivrea (Italia) como un proyecto para estudiantes. Sus creadores, Massimo Banzi, David Cuartielles, Tom Igoe, Gianluca Martino y David Mellis, buscaban una herramienta sencilla y económica para que los estudiantes pudieran crear proyectos de electrónica sin necesidad de conocimientos avanzados.

Desde entonces, Arduino ha evolucionado considerablemente:

  • 2005: Primera placa Arduino (Serial)
  • 2008: Arduino Duemilanove, una de las primeras placas ampliamente adoptadas
  • 2010: Arduino Uno, que se convirtió en la placa estándar
  • 2012: Arduino Leonardo, con capacidades USB mejoradas
  • 2014: Arduino Yún, con conectividad WiFi
  • 2015: Arduino MKR1000, enfocada en IoT
  • 2018: Arduino Nano 33 IoT, con Bluetooth y WiFi en formato compacto
  • 2020: Arduino Portenta H7, para aplicaciones industriales

Programación de Arduino

La programación de Arduino se realiza mediante el entorno de desarrollo (IDE) de Arduino, que proporciona un editor de código, un compilador y herramientas para cargar el programa en la placa.

Entorno de Desarrollo (IDE)

El IDE de Arduino es un software multiplataforma (Windows, macOS, Linux) basado en el lenguaje de programación Processing. Facilita la escritura, compilación y carga de programas en las placas Arduino.

Características principales:
  • Editor de código: Permite escribir y editar el código fuente
  • Compilador: Traduce el código fuente a código máquina
  • Cargador: Transfiere el código máquina a la placa Arduino
  • Bibliotecas: Proporcionan funciones predefinidas para facilitar la programación
  • Ejemplos: Incluye ejemplos de código para aprender y experimentar

Lenguaje de Programación

El lenguaje de programación de Arduino se basa en C/C++, pero con algunas simplificaciones y extensiones para facilitar el uso de la electrónica. Los programas de Arduino se llaman "sketches".

Estructura básica de un sketch:
Código Arduino 
void setup() {
  // Inicialización del programa
}

void loop() {
  // Bucle principal del programa
}

La función setup() se ejecuta una sola vez al inicio del programa. Se utiliza para inicializar variables, configurar pines y establecer la comunicación serial.

La función loop() se ejecuta continuamente después de la función setup(). Contiene el código principal del programa.

Funciones Comunes

Arduino proporciona una serie de funciones predefinidas para facilitar la programación de la electrónica:

  • pinMode(pin, mode): Configura un pin como entrada o salida
  • digitalWrite(pin, value): Escribe un valor digital (HIGH o LOW) en un pin
  • digitalRead(pin): Lee un valor digital de un pin
  • analogWrite(pin, value): Escribe un valor analógico (PWM) en un pin
  • analogRead(pin): Lee un valor analógico de un pin
  • delay(ms): Pausa la ejecución del programa durante un número de milisegundos
  • Serial.begin(baud): Inicializa la comunicación serial
  • Serial.print(data): Envía datos a través de la comunicación serial

Ejemplos de Código

A continuación, se presentan algunos ejemplos de código para ilustrar el uso de Arduino en proyectos sencillos.

Ejemplo 1: Blink (Parpadeo de un LED)

Este ejemplo hace parpadear un LED conectado al pin 13 de la placa Arduino.

Código Arduino 
int ledPin = 13; // Pin del LED

void setup() {
  pinMode(ledPin, OUTPUT); // Configura el pin como salida
}

void loop() {
  digitalWrite(ledPin, HIGH); // Enciende el LED
  delay(1000); // Espera 1 segundo
  digitalWrite(ledPin, LOW); // Apaga el LED
  delay(1000); // Espera 1 segundo
}

El programa define el pin 13 como salida y luego, en el bucle principal, enciende y apaga el LED cada segundo.

Ejemplo 2: Lectura de un Sensor Analógico

Este ejemplo lee el valor de un sensor analógico conectado al pin A0 de la placa Arduino y lo muestra a través de la comunicación serial.

Código Arduino 
int sensorPin = A0; // Pin del sensor

void setup() {
  Serial.begin(9600); // Inicializa la comunicación serial
}

void loop() {
  int sensorValue = analogRead(sensorPin); // Lee el valor del sensor
  Serial.print("Valor del sensor: ");
  Serial.println(sensorValue); // Muestra el valor en el monitor serial
  delay(100); // Espera 100 milisegundos
}

El programa inicializa la comunicación serial y luego, en el bucle principal, lee el valor del sensor analógico y lo muestra en el monitor serial.
